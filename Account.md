# Add user and build an env

## Create an account

```bash
# change default shell from /bin/sh to /bin/bash
sudo useradd -D -s /bin/bash

# create new account
sudo useradd -m -s /bin/bash wzy01
# -m: create /home/wzy01
# -s: use /bin/bash as the default shell

# set new passwd
sudo passwd wzy01
# type 2 times
```
